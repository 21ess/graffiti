# Game Systems & Data Guidelines

> Best practices for game systems, data management, and backend architecture in this project.

---

## Overview

This directory contains guidelines for the **systems & data layer** — autoloads, resources, data persistence, error handling, and game infrastructure. This is the "backend" of the game: systems that support gameplay without being gameplay themselves.
